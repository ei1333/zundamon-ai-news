About

このページについて

「ずんだもん1分AIニュース」は、公開情報をもとに注目トピックを短く整理し、音声でもさっと確認できるようにまとめたニュースページです。

なにをしているページか

毎回ひとつのテーマを選び、公開情報から3本前後の話題に絞って要点をまとめています。長文を追わなくても、トップページと音声だけでその日の流れを掴みやすい構成を目指しています。

どうやって作っているか

1. 公開情報を収集

各テーマごとに公開されているニュースや告知を確認し、取り上げる候補を絞ります。

2. OpenClaw で整形・更新

OpenClaw を使ってページの文面調整、構成変更、更新作業を進めています。

3. GitHub へ push

HTML / CSS や各日のページを GitHub リポジトリへ反映します。

4. GitHub Pages で公開

push 後に GitHub Pages 側へ反映され、ブラウザから閲覧できるようになります。

アーキテクチャ概要

  • フロントエンドは静的な HTML / CSS ベースです。
  • 更新フローには OpenClaw を利用しています。
  • ソース管理は GitHub、公開は GitHub Pages です。
  • 各回の詳細ページとトップページを行き来しやすい構成にしています。

出典の見方

各日の詳細ページでは、3本前後のトピックごとに参照した公開情報へのリンクを「出典」として掲載しています。気になる話題はトップページの「詳細を見る」から移動し、元記事や公式発表をたどれるようにしています。

補足

内容は公開情報をもとに整理しています。ページ構成や表現は継続的に調整することがあります。
